黑名單

DNSBL Information - Spam Database Lookup

Blocklist Removal Center - The Spamhaus Project

解決方案

  1. 更換 Mail Server 主機 IP (workaround?)
  2. DNS Server 設定MX
  3. 設定SPF
  4. 設定 DomainKey（DKIM）
  5. 設定 DMARC
  6. 開通25 Port

建製 mail 注意的事情

  1. 整理發信來源IP
  2. 調整DNS，A、PTR、MX、TXT record、DKIM
  3. 向各Email服務商提出申請
  4. 發信機主機名稱
  5. LOG解析

相關DNS Resource

調整DNS
A：正解
PTR：反解
MX：郵件交換紀錄
TXT record：SPF
DKIM

MX：讓DNS能正常解析
SPF：是一套電子郵件驗證機制，可指定哪些郵件伺服器有權為您的網域傳送電子郵件。
DKIM：會為每一封郵件加上數位簽名，幫助收件伺服器確認郵件並非偽造，而且在傳輸過程中未遭到竄改。
