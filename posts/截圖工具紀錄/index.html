最近朋友問我有甚麼好用截圖工具，整理幾款常用的選項，從免費到付費都有，適合不同需求的使用者。

Windows 內建截圖工具

在安裝第三方工具前，先了解 Windows 內建的截圖功能：

Print Screen 按鍵

  • PrtScn：截取整個螢幕，複製到剪貼簿
  • Alt + PrtScn：截取目前視窗，複製到剪貼簿
  • Win + PrtScn：截取整個螢幕並自動存檔到「圖片 > 螢幕擷取畫面」資料夾

Snipping Tool（剪取工具）

  • Windows 10/11 內建，可按 Win + Shift + S 快速呼叫
  • 支援矩形、任意形狀、視窗、全螢幕等截圖模式
  • 可以直接標注、裁切後儲存
  • Windows 11 的 Snipping Tool 已支援文字擷取（OCR）

內建工具適合偶爾截圖，若需要更多功能（標注、上傳、自動化），推薦以下第三方工具。

Greenshot（免費推薦）

官網：https://getgreenshot.org/

Greenshot 是 Windows 平台最推薦的免費截圖工具，功能完整且輕量。

主要功能

  • 多種截圖模式：全螢幕、視窗、自訂區域、捲動截圖
  • 快捷鍵自訂：預設 Print Screen 觸發，按 Capture Region 框選區域
  • 內建圖片編輯器：可加入箭頭、文字、馬賽克、框線、螢光筆等標注
  • OCR 文字辨識：透過 Greenshot 外掛支援 OCR（需安裝 Office 或搭配外掛）
  • 多種輸出方式：存檔、複製到剪貼簿、開啟圖片編輯器、直接上傳到 Imgur/Flickr 等服務
  • 外掛擴充：支援上傳到 Jira、Confluence、Dropbox 等平台

適合對象

日常開發工作、需要快速截圖並標注後分享給同事的工程師，是免費方案中的最佳選擇。

Snagit（付費專業）

官網：https://www.techsmith.com/screen-capture.html

Snagit 是 TechSmith 出品的專業截圖工具，功能強大，適合需要製作教學文件或產品說明的使用者。

主要功能

  • 專業圖片標注：豐富的標注工具，支援步驟數字、標注箭頭、說明框等
  • GIF 錄製：可錄製螢幕操作並輸出為 GIF 動圖，適合製作操作示範
  • 影片錄製：支援螢幕錄影，可搭配麥克風錄音
  • 捲動截圖：完整截取長網頁或文件
  • 圖片整合：可將多張截圖合併為一個說明文件
  • TechSmith Screencast 雲端儲存：截圖可直接上傳到雲端並分享連結

適合對象

需要製作精緻的產品文件、使用手冊或教學內容的人，尤其是技術文件撰寫者。

ShareX（免費進階）

官網：https://getsharex.com/

ShareX 是功能最強大的免費截圖工具，幾乎所有進階功能都有，適合對自動化工作流程有需求的使用者。

主要功能

  • 截圖 + 錄影：支援全螢幕、視窗、區域截圖，以及 GIF 和影片錄製
  • OCR 文字辨識：截圖後可直接辨識圖片中的文字並複製
  • 雲端分享：內建支援上傳到 Imgur、Google Drive、Dropbox、FTP、OneDrive 等 30+ 個目的地
  • 自動化工作流程（Task）：截圖後可自動執行一系列動作，例如：截圖 → 加浮水印 → 上傳 → 複製連結到剪貼簿
  • 螢幕顏色擷取：可取得螢幕上任何位置的色碼（HEX/RGB）
  • 工具集：內建圖片編輯器、圖片裁切器、影像轉換器、QR Code 產生器等

適合對象

需要頻繁截圖並搭配自動化上傳流程的開發者、技術部落客，或是對截圖工具有深度需求的進階使用者。
