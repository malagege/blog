MySQL查詢進行中程序狀況，並刪掉lock程序

malagege included in MySQL
 2018-05-17  34 words   One minute 
Contents

因為網域關係，平常都是使用phpmyadmin預做設定
有時候下比較差的SQL，就不能執行中斷指令
phpmyadmin就會卡住….
剛好發現用cli模式登入mysql
可以做刪除程序

1

mysql -h ip -u xxxx -p

就可以登入mysql資料庫裡面
只要在不打;下
無法送出指令

1

show processlist;

可以看到是否有lock程序

使用kill可以做刪除動作

1

kill id ;

發現在不用phpmyadmin時候，cli可以中斷程序
phpmyadmin送出去，就像變了心女朋友一樣回不來了不能中斷
只是無法修改是一行打做的東西
