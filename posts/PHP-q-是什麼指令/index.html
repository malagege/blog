 $ php-cgi p.php

    X-Powered-By: PHP/5.5.18
    Content-type: text/html
    hello world

You can suppress these headers:
$ php-cgi -q p.php
    hello world
The php executable in recent versions of PHP doesn't need this. Using -q or not using -q gives the same result:
    $ php -q p.php
hello world
$ php p.php
hello world
