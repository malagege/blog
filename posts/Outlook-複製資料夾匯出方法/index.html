Outlook 複製資料夾匯出方法

Outlook 的 .pst 檔案是「個人資料夾」格式，可以用來備份郵件、匯出資料夾結構，以及在不同電腦或帳戶之間遷移郵件。本文說明如何匯出、匯入，以及複製空資料夾結構。

.pst 檔案說明

.pst（Personal Storage Table）是 Outlook 儲存郵件、行事曆、聯絡人的本機檔案格式：

  • 大小限制：
    • Outlook 2003 以前：2GB 限制
    • Outlook 2007 以後：預設上限 50GB（可調整）
  • 適用帳戶：POP3 帳戶本地儲存、Exchange 帳戶封存
  • 注意：Exchange Online（Office 365）和 IMAP 帳戶的郵件儲存在伺服器，匯出 .pst 是在本機製作備份副本

匯出郵件到 .pst 檔案

Outlook 2013 / 2016 / 2019 / 365

  1. 點選「檔案」→「開啟和匯出」→「匯入/匯出」
  2. 選擇「匯出至檔案」→「下一步」
  3. 選擇「Outlook 資料檔 (.pst)」→「下一步」
  4. 選擇要匯出的資料夾（可選擇「包含子資料夾」）
  5. 指定儲存路徑和檔名
  6. 選擇重複項目的處理方式 → 「完成」

若要備份特定時間範圍內的郵件，可在步驟 4 點選「篩選器」設定日期範圍。

只複製資料夾結構（不含郵件）

當需要在新帳戶中重建相同的資料夾架構，而不需要郵件內容時：

方法一：使用「封存」功能

  1. 點選「檔案」→「資訊」→「清理工具」→「封存」
  2. 選擇來源資料夾
  3. 將「封存日期早於」設定為比所有郵件都早的日期（例如 1990/01/01）
  4. 指定輸出 .pst 路徑 → 「確定」

由於日期設定比任何郵件都早，封存結果只包含資料夾結構，不含任何郵件。

方法二：Outlook 2010 的操作

  1. 點選「檔案」→「資訊」→「清理工具」→「封存」
  2. 選擇「封存此資料夾及其下的所有子資料夾」
  3. 設定封存日期早於所有信件的日期
  4. 選擇目的地 .pst 路徑

匯入 .pst 檔案

匯入到現有帳戶

  1. 點選「檔案」→「開啟和匯出」→「匯入/匯出」
  2. 選擇「從另一程式或檔案匯入」→「下一步」
  3. 選擇「Outlook 資料檔 (.pst)」→「下一步」
  4. 選擇要匯入的 .pst 檔案路徑
  5. 選擇目標資料夾位置 → 「完成」

掛載 .pst 為附加資料夾

若只是要查看而不匯入，可以將 .pst 掛載：

  1. 「檔案」→「開啟和匯出」→「開啟 Outlook 資料檔案」
  2. 選擇 .pst 檔案，它會出現在左側資料夾清單中
  3. 可直接瀏覽或拖拉郵件

Exchange 與 IMAP 帳戶的差異

帳戶類型 郵件存放位置 匯出說明
POP3 本機 .pst 完全匯出
Exchange（公司伺服器） 伺服器 匯出為本機備份副本
Exchange Online（Office 365） 雲端 匯出副本，原件仍在雲端
IMAP（Gmail 等） 伺服器 只匯出已下載到本機的郵件

若使用 Exchange 帳戶，管理員也可以從 Exchange Admin Center 進行更完整的郵箱匯出（eDiscovery 功能）。

常見問題

Q：匯出後 .pst 檔案很小，郵件不完整？
A：IMAP 帳戶可能未完整下載所有郵件。先在各資料夾開啟「下載此資料夾中的所有郵件」，再重新匯出。

Q：.pst 損壞無法開啟？
A：使用 Office 安裝目錄中的 scanpst.exe（收件匣修復工具）修復損壞的 .pst 檔案。
