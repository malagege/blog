在 Linux 設定 方法

其實這時候很簡單，調整時區就能解決問題!!

Crontab 能不能調整時區?
答案可以：
CentOS7 下 Crontab 时区问题 | MYF

ntpdate -u tock.stdtime.gov.tw
timedatectl set-timezone Asia/Taipei
timedatectl set-local-rtc 0 # Set Real-Time Clock
cp -pf /usr/share/zoneinfo/Asia/Taipei /etc/localtime
