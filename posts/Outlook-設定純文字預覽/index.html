Outlook 設定純文字預覽

公司常常寄資安演練信，不小心點開就會被記錄。改用純文字模式開信，可以有效防止這類追蹤。

為什麼純文字模式可以防止追蹤？

HTML 格式的電子郵件可以內嵌外部圖片（包括 1x1 像素的追蹤像素 / Tracking Pixel）。當你開啟郵件時，Outlook 會自動載入這些圖片，對方的伺服器就能收到請求，得知：

  • 你的 IP 位址
  • 開信時間
  • 使用的郵件客戶端

啟用純文字模式後，郵件中的 HTML 和圖片都不會被載入，追蹤像素自然失效。

設定步驟：所有郵件預設以純文字讀取

  1. 開啟 Outlook
  2. 點擊上方選單列的 「檔案」（File）
  3. 選擇 「選項」（Options）
  4. 在左側選單點選 「信任中心」（Trust Center）
  5. 點擊右側的 「信任中心設定…」（Trust Center Settings）
  6. 選擇 「電子郵件安全性」（Email Security）
  7. 勾選 「以純文字格式讀取所有標準郵件」（Read all standard mail in plain text）
  8. 按 「確定」 儲存

臨時切換單封郵件的顯示格式

如果已啟用純文字模式，但偶爾需要查看特定郵件的 HTML 格式：

  1. 開啟該封郵件
  2. 點擊郵件頂端出現的黃色資訊列（說明「此郵件已轉換為純文字格式」）
  3. 點擊 「按一下這裡以用 HTML 格式顯示」

這樣只有這封郵件會以 HTML 顯示，其他設定不變。

其他 Outlook 資安設定建議

停用自動下載外部圖片

  1. 信任中心 → 自動下載
  2. 勾選 「不要在標準 HTML 電子郵件訊息或 RSS 項目中自動下載圖片」

停用自動預覽

即使沒有開啟郵件，預覽窗格也可能觸發追蹤像素。建議：

  1. 「檢視」→「讀取窗格」→ 選擇「關閉」

或設定為只有點擊後才載入圖片。

啟用垃圾郵件篩選

  1. 在收件匣點擊右鍵 → 「垃圾郵件」→「垃圾郵件選項」
  2. 將安全等級設為 「高」

小結

設定 位置 效果
純文字閱讀模式 信任中心 → 電子郵件安全性 防止追蹤像素、停止載入 HTML
停用自動下載圖片 信任中心 → 自動下載 不自動向外部伺服器發送請求
關閉預覽窗格 檢視 → 讀取窗格 避免在未點開郵件時觸發追蹤
