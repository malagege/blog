常常我們在測試 DB 環境
不會馬上上到正式環境
但都是分開改 alter table
之前有在 Dbeaver 有 migrate table 功能
但大致上我們工程師沒有 DB 的權限
要如何快速產生 alter table 語法呢?

其實之前有寫過 MySQL 比較兩個資料庫方法 | 程式狂想筆記
但一樣沒有產生 alter table sql 語法…
裡面的範例也沒有使用過
今天我發現 github 裡面有一個 schemalex
可以輕鬆產生 alter table 語法
