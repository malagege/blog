因為工作需求要專案檔案差異清單，要怎麼快速整理清單真的讓我想了一下。最後發現 Winmerge 就能做到簡單程式差異比對。

tree mode

merge - Get recursive list of different files in Winmerge (without folder tree/structure) - Super User

可以方便複製檔案清單。可跳過上層資料夾資料。

開啟/關閉 顯示相同相目,顯示差異項目

選擇「顯示相同項目」後，原本綠色的圖標會顯示/隱藏。這樣看檔案(附製檔案也很方便)。

顯示差異項目也是跟上面差不多。
