1.创建临时环境变量：

windows上命令行输入：

set GIT_SSL_NO_VERIFY=true git clone

Linux下：

env GIT_SSL_NO_VERIFY=true git push

但我的 window 的 git bash 上面指令都不管用
不過我後來找到一篇用

1

export GIT_SSL_NO_VERIFY=1
