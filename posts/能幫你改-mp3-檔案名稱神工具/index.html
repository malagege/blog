整理大量音樂檔案時，MP3 的 ID3 標籤（歌手、專輯、曲目等資訊）不完整或檔案命名混亂是常見問題。本文介紹兩款好用的工具，幫助你批量整理音樂標籤和檔案名稱。

MusicBrainz Picard（強力推薦）

MusicBrainz Picard 是一款開源的音樂標籤管理工具，能自動透過音訊指紋（AcoustID）辨識歌曲，從 MusicBrainz 音樂資料庫抓取完整的 ID3 標籤。

下載地址

主要功能

  • 音訊指紋辨識：即使檔案名稱完全錯誤，也能透過音訊內容識別歌曲身份
  • 從 MusicBrainz 自動填寫標籤：歌手、專輯、發行年份、曲目編號等
  • 批量重命名：根據 ID3 標籤自動重命名檔案
  • 預設分類：依歌手 / 專輯自動建立資料夾結構

使用步驟

  1. 將音樂檔案拖曳到 Picard 視窗
  2. 點選 Lookup（查詢）或 Scan（掃描音訊指紋）
  3. Picard 會自動比對 MusicBrainz 資料庫
  4. 確認比對結果後，點選 Save（儲存標籤）

自訂命名格式

在 Options → File Naming 可設定檔案命名規則：

1

%artist% - %title%

預設格式會依 歌手名稱/專輯名稱/曲目 建立資料夾，整理效果非常好。

注意：預設會將音樂移動到你在設定中指定的目標音樂資料夾，確認設定再執行。

適合場景

  • 整批 MP3 標籤都不完整或亂七八糟
  • 需要從頭建立整齊的音樂資料庫
  • 希望按歌手 / 專輯自動分類

Mp3tag（Windows 手動整理神器）

Mp3tag 是一款 Windows 平台老牌的音樂標籤編輯器，提供更細緻的手動控制。

主要功能

  • 批量編輯 ID3 標籤：同時修改多個檔案的歌手、專輯等欄位
  • 從標籤重命名檔案：用 %artist% - %title% 等格式批量重命名
  • 從檔名填入標籤：解析檔名中的資訊（如 01 - 歌名.mp3）反填標籤
  • 從網路取得標籤：支援 Discogs、MusicBrainz 等來源
  • 正規表達式替換：批量取代標籤內容中的特定文字
  • 匯出 / 匯入：支援 CSV 格式批量處理

常用操作

從標籤批量重命名檔案：

  1. 選取所有要重命名的檔案
  2. 點選選單 Convert → Tag - Filename
  3. 輸入命名格式，例如：%track% - %title%
  4. 點 OK 執行

從檔案名稱填入標籤：

  1. 選取檔案
  2. Convert → Filename - Tag
  3. 輸入解析格式，例如：%artist% - %title%（對應檔名 歌手 - 歌名.mp3）

正規表達式批量取代：

  1. Convert → Text operations
  2. 選擇要修改的欄位（如 Title）
  3. 使用正規表達式定義搜尋和取代規則

適合場景

  • 需要精細控制每個標籤欄位
  • 已有整齊的命名規則，要轉換成標籤
  • 批量清理特定文字（如去除標題中多餘的空格、括號）

兩款工具比較

功能 MusicBrainz Picard Mp3tag
自動辨識歌曲 ✅（音訊指紋） ❌
自動取得標籤 ✅（MusicBrainz） ✅（多來源）
手動編輯標籤 ✅ ✅
批量重命名 ✅ ✅
正規表達式 限制較多 ✅
平台支援 跨平台 Windows
授權 開源免費 免費（捐款）

整理建議：先用 MusicBrainz Picard 自動識別並填入標籤，再用 Mp3tag 進行細部調整。

FlexibleRenamer

另一個選項是 FlexibleRenamer，是一款通用的批量重命名工具，不只限於音樂檔案，支援正規表達式和多種重命名規則。對於只需要重命名（不需要修改 ID3 標籤）的情境很適用。
